# TaskFlow Engine

A minimal workflow engine for defining tasks and executing them based on dependencies.

## Features

- Define tasks with Python callables
- Define workflows with dependency graphs
- Execute workflows and track task status
- Simple example included

## Quick start

```bash
git clone https://github.com/<your-username>/taskflow-engine.git
cd taskflow-engine
pip install -r requirements.txt
python examples/simple_workflow.py
